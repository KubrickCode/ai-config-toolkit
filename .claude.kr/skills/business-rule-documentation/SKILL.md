## 문서 관리

### 문서 위치

```
project-root/
├── docs/
│   ├── work/          # 작업 가이드라인 (작업 완료 후 삭제)
│   └── domain/        # 비즈니스 로직 문서 (지속 유지)
├── WORK_SUMMARY.md    # 작업 보고서 (확인 후 즉시 삭제)
└── README.md
```

### 생명주기

- **작업 가이드라인:** 작업 완료 후 삭제
- **작업 보고서:** 확인 즉시 삭제
- **비즈니스 로직 문서:** 코드와 함께 유지보수
